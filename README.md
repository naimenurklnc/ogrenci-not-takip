#  Öğrenci Not Takip Sistemi

Python ile yazılmış, JSON tabanlı öğrenci not takip uygulaması.

## Özellikler
- Öğrenci ekleme ve silme
- Vize / Final notu girişi
- Otomatik ortalama hesaplama (Vize %40, Final %60)
- Veriler JSON dosyasına kaydedilir

## Kullanım
```bash
python not_takip.py
```

## Kullanılan Teknolojiler
- Python 3
- json modülü
- os modülü
